Простое приложение-визитка на Jetpack Compose. Отображает информацию о разработчике с контактными данными.

## 📱 О проекте

Приложение показывает:
- Фото профиля
- Имя и должность
- Контактную информацию (телефон, Telegram, email)

## 🛠 Технологии

- **Kotlin** — язык программирования
- **Jetpack Compose** — UI toolkit
- **Material Design 3** — дизайн-система
